# 패키지

**클래스들을 묶어 놓은 단위**

물리적으로는 패키지명 = 폴더명

*패키지 형식: 상위패키지.하위패키지.클래스명*

## 접근제한자
```java
public class Dog{

}
```

### public
접근 제한 없음(외부 패키지에서도 접근 가능)
### protected
같은 패키지에서 자식클래스만 접근 가능
### default(package)
같은 패키지에서 접근 가능(*문구 생략 가능*)
### private
같은 클래스에서만 접근 가능

은닉화(캡슐화)를 할 때 주로 사용

*외부에서 접근하려면 Getter() 또는 Setter() 메소드가 필요*

*static 변수의 경우 instance를 사용*
