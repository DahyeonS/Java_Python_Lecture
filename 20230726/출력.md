# 출력
### System.out.println()
괄호 안 내용을 출력 후 줄바꿈
### System.out.print()
괄호 안 내용을 출력
### System.out.printf()
```
System.out.printf("형식", 값)
```
괄호 안 내용을 지정된 형식으로 출력
> #### 출력 형식
> - %d -> *10진수 정수*
> - %o -> *8진수 정수*
> - %x -> *16진수 정수*
> - %f -> *실수*
> - %e -> *지수*
> - %c -> *문자*
> - %s -> *문자열*
> - %n -> *줄바꿈(printf에서만 사용)*
>   
> (번외) \n -> 줄바꿈 **(printf 밖에서도 사용가능)**
