# 리눅스 사용법

- sudo - 관리자 권한으로 실행
- cat - 파일 내용 출력
- ls - 현재 디렉토리 내용 출력(-a는 숨김 파일 포함 출력, -l은 자세히 출력)
- clear - 화면 지우기
- touch 파일이름 - 새 파일 생성
- find - 특정 파일이나 디렉토리 검색
- reboot - 재시작
- list - 목록 확인
- source 또는 . - 재컴파일
- mv - 이름 변경 및 위치 이동
- ln - 캡션(별명) 추가
- useradd - 사용자 추가

## vi 명령어
### 명령 모드
- dd - 한 줄 삭제
- yy - 한 줄 붙여넣

### 입력 모드
- i - 현재 커서에서 입력 모드 전환
- a - 현재 커서 다음부터 입력 모드 전환
- o - 다음 줄부터 입력 모드 전환
- Esc - 입력 모드 종료 및 명령 모드 전환

### 마지막 행 모드
- :q - 저장하지 않고 종료
- :q! - 저장하지 않고 강제종료
- :wq - 저장 후 종료
- :wq! - 저장 후 강제종료
