# 제어문
## 조건문
### if
주어진 조건식이 true일 때, **if** 명령문을 실행

주어진 조건식이 false일 경우, 다음으로 제시된 *else if* 조건식이 true면 명령문을 실행

모두 false일 경우 *else* 명령문을 실행
```java
if (x == y) {
  System.out.println("x와 y는 같다.");
} else if (x > y) {
  System.out.println("x는 y보다 크다.");
} else {
  System.out.println("x는 y보다 작다.");
}
```
