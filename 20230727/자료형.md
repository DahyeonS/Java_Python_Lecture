# 자료형
## 기본형
### 숫자형
> #### 정수
> - byte
> - short
> - **int** - 2^32(약 1억) 미만의 정수
> ```java
> int i = 1;
> ```
> - **long** - 2^32(약 1억) 이상의 정수
> ```
> long l = 1L;
> long l = 1l;
> ```

> #### 실수
> - **float** - 4바이트 실수
> ```java
> float f = 3.14F;
> float f = 3.14f;
> ```
> - **double** - 8바이트 실수
> ```java
> double d = 3.14;
> ```
### 논리형
true 또는 false
```java
boolean b = false;
boolean b = 1 + 1 == 2; // true
boolean b = 1 + 1 != 2; // false
```
