# 연산자
## 산술 연산자
### 덧셈
**x + y**
```java
int x = 1;
int y = 1;

x + y; // 2
```
### 뺄셈
**x - y**
```java
int x = 3;
int y = 2;

x - y; // 1
```
### 곱셈
**x * y**
```java
int x = 3;
int y = 3;

x * y; //9
```
### 나눗셈
**x / y**
```java
int x = 8;
int y = 2;

x / y; // 4
```
```java
int x = 5;
iny y = 2;

(double)x / y; // 2.5
```
### *나머지*
**x % y**
```java
int x = 10;
int y = 3;

x % y; // 1
```
